Eventex events app
==============================

Events app using Eventbrite API and built with ReactJS + Redux

## Features

- Webpack configuration for production
- Use of ES6
- Firebase Backend
- Authentication

## Installation

Clone this repo
```
git clone https://github.com/neil-mills/eventex.git
cd sportz
```

Install all the dependencies
```
npm install
```

Run the app
```
npm run start
```
